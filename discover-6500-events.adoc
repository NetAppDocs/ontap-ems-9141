== discover.6500.supported
*Severity*::
NOTICE
*Description*::
This message occurs when discovery of a 6500N FC-to-SAS bridge is enabled.
*Corrective Action*::
(None).
*Syslog Message*::
FC-to-SAS bridge ATTO 6500N discovery is enabled.
*Parameters*::
(None).

*Severity*::
NOTICE
*Description*::

*Corrective Action*::
(None).
*Syslog Message*::
FC-to-SAS bridge ATTO 6500N discovery is disabled.
*Parameters*::
(None).
